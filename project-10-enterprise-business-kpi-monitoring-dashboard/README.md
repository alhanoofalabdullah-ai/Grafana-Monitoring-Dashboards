# Project 10 – Enterprise Business KPI Monitoring Dashboard

Enterprise KPI Dashboard for executive reporting using Grafana, Prometheus, Spring Boot, Docker, and Business Analytics.

---

## Features

- Revenue Monitoring
- Project Performance Tracking
- SLA Monitoring
- Customer Satisfaction Tracking
- Service Availability Monitoring
- Executive KPI Dashboard
- Alert Management
- Business Analytics

---

## Technology Stack

- Java 21
- Spring Boot
- Grafana
- Prometheus
- Docker
- Maven

---

## Services

Grafana

http://localhost:3000

Prometheus

http://localhost:9090

Business KPI API

http://localhost:8080/api/kpi
