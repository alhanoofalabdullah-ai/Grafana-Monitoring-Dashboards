# Project 11 – Enterprise NOC Monitoring Dashboard

Enterprise Network Operations Center dashboard using Grafana, Prometheus, Spring Boot, Docker, and monitoring APIs.

## Features

- Infrastructure Health Monitoring
- Network Availability Tracking
- Incident Monitoring
- Service Uptime Tracking
- Critical Alerts
- NOC Executive Dashboard
- Real-Time Operations Visibility

## Technology Stack

- Java 21
- Spring Boot
- Prometheus
- Grafana
- Docker
- Maven

## Services

Grafana: http://localhost:3000

Prometheus: http://localhost:9090

NOC API: http://localhost:8080/api/noc
