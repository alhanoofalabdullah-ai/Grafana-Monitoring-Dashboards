# Project 16 – ELK Stack Monitoring Dashboard

Enterprise Log Analytics and Monitoring Platform using Elasticsearch, Logstash, Kibana, Grafana, Prometheus, Spring Boot, and Docker.

## Features

- Centralized Log Collection
- Log Analytics
- Error Detection
- Application Monitoring
- Kibana Dashboards
- Grafana Dashboards
- Elasticsearch Monitoring
- Logstash Pipeline Monitoring

## Technology Stack

- Java 21
- Spring Boot
- Elasticsearch
- Logstash
- Kibana
- Prometheus
- Grafana
- Docker
