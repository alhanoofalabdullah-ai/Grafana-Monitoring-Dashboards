# Project 08 – Cloud Infrastructure Monitoring Dashboard

Enterprise Cloud Infrastructure Monitoring Platform using Grafana, Prometheus, Docker, and Spring Boot.

---

## Features

- Cloud VM Monitoring
- CPU Utilization Tracking
- Memory Monitoring
- Storage Monitoring
- Cloud Resource Monitoring
- Health Status Monitoring
- Alert Management
- Grafana Dashboards

---

## Technologies

- Java 21
- Spring Boot
- Grafana
- Prometheus
- Docker
- Maven

---

## Services

Grafana

http://localhost:3000

Prometheus

http://localhost:9090

Cloud Monitoring API

http://localhost:8080/api/cloud
