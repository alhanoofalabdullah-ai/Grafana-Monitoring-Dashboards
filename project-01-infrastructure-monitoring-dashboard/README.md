# Project 01 – Enterprise Infrastructure Monitoring Center

Enterprise monitoring platform using:

- Grafana
- Prometheus
- Spring Boot
- Docker
- Node Exporter

The platform provides centralized monitoring and observability for enterprise infrastructure.

---

## Features

- CPU Monitoring
- Memory Monitoring
- Disk Monitoring
- Server Health Monitoring
- Prometheus Alerts
- Grafana Dashboards
- REST API Metrics
- Docker Deployment

---

## Architecture

Server

↓

Node Exporter

↓

Prometheus

↓

Spring Boot Monitoring API

↓

Grafana Dashboard

## Technologies

- Java 21
- Spring Boot
