# Project 03 – Database Monitoring Dashboard

Enterprise database monitoring dashboard using Grafana, Prometheus, MySQL Exporter, Docker, and Spring Boot.

## Overview

This project monitors database health, availability, connections, queries, and storage indicators.

## Features

- MySQL Health Monitoring
- Active Connections Tracking
- Query Volume Monitoring
- Database Uptime
- Prometheus Alerts
- Grafana Dashboard
- Spring Boot Monitoring API
- Docker Deployment

## Technologies

- Java 21
- Spring Boot
- MySQL
- Prometheus
- Grafana
- MySQL Exporter
- Docker

## Services

Grafana: http://localhost:3000

Prometheus: http://localhost:9090

Monitoring API: http://localhost:8080/api/database-metrics

MySQL: localhost:3306
