# Project 12 – SLA Monitoring Dashboard

Enterprise SLA Monitoring Dashboard using Grafana, Prometheus, Spring Boot, Docker, and REST APIs.

## Features

- SLA Compliance Monitoring
- Service Availability Tracking
- Incident Breach Tracking
- Response Time Monitoring
- Resolution Time Monitoring
- Executive SLA Dashboard
- Prometheus Alerts
- Docker Deployment

## Technology Stack

- Java 21
- Spring Boot
- Prometheus
- Grafana
- Docker
- Maven

## Services

Grafana: http://localhost:3000

Prometheus: http://localhost:9090

SLA API: http://localhost:8080/api/sla
