# Project 13 – Application Performance Monitoring Dashboard

Enterprise APM Dashboard using Grafana, Prometheus, Spring Boot, Docker, and Application Performance Metrics.

## Features

- Application Response Time Monitoring
- Request Throughput Monitoring
- Error Rate Monitoring
- JVM Monitoring
- Memory Consumption Monitoring
- CPU Utilization Monitoring
- Real-Time Performance Analytics
- Executive Dashboard

## Technology Stack

- Java 21
- Spring Boot
- Prometheus
- Grafana
- Docker
- Micrometer
- Maven

## Services

Grafana

http://localhost:3000

Prometheus

http://localhost:9090

APM API

http://localhost:8080/api/apm
