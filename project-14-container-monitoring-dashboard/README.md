# Project 14 – Container Monitoring Dashboard

Enterprise Container Monitoring Dashboard using Grafana, Prometheus, cAdvisor, Spring Boot, and Docker.

---

## Features

- Docker Container Monitoring
- CPU Usage Monitoring
- Memory Usage Monitoring
- Container Health Tracking
- Running Containers Dashboard
- Container Resource Analytics
- Alert Management
- Real-Time Monitoring

---

## Technology Stack

- Java 21
- Spring Boot
- Docker
- cAdvisor
- Prometheus
- Grafana
- Maven

---

## Services

Grafana

http://localhost:3000

Prometheus

http://localhost:9090

Container API

http://localhost:8080/api/containers

cAdvisor
