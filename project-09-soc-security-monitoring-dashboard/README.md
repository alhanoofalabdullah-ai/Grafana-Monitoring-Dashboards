# Project 09 – SOC Security Monitoring Dashboard

Enterprise Security Operations Center (SOC) Dashboard using Grafana, Prometheus, Spring Boot, and Docker.

---

## Features

- Security Event Monitoring
- Failed Login Detection
- Brute Force Detection
- Threat Monitoring
- Critical Alert Dashboard
- Incident Tracking
- Security Analytics
- Real-Time Monitoring

---

## Technology Stack

- Java 21
- Spring Boot
- Grafana
- Prometheus
- Docker
- Maven

---

## Services

Grafana

http://localhost:3000

Prometheus

http://localhost:9090

Security Monitoring API

http://localhost:8080/api/security
