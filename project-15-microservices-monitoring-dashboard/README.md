# Project 15 – Microservices Monitoring Dashboard

Enterprise monitoring solution for Microservices Architecture using Grafana, Prometheus, Spring Boot, Docker, and API Gateway monitoring.

---

## Features

- API Gateway Monitoring
- User Service Monitoring
- Order Service Monitoring
- Service Health Tracking
- Inter-Service Communication Monitoring
- Response Time Monitoring
- Error Rate Tracking
- Distributed System Dashboard

---

## Technology Stack

- Java 21
- Spring Boot
- Prometheus
- Grafana
- Docker
- REST APIs
- Microservices

---

## Services

Grafana

http://localhost:3000

Prometheus

http://localhost:9090

Monitoring API

http://localhost:8080/api/microservices
