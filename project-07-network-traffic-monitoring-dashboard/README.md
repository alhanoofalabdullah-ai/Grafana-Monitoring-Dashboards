# Project 07 – Network Traffic Monitoring Dashboard

Enterprise network traffic monitoring solution using Grafana, Prometheus, Spring Boot, Docker, and network telemetry metrics.

---

## Features

- Network Traffic Monitoring
- Bandwidth Usage Monitoring
- Packet Analysis
- Network Latency Monitoring
- Interface Monitoring
- Alert Management
- Grafana Dashboards
- REST API Metrics

---

## Technology Stack

- Java 21
- Spring Boot
- Grafana
- Prometheus
- Docker
- Maven

---

## Services

Grafana

http://localhost:3000

Prometheus

http://localhost:9090

Network Monitoring API

http://localhost:8080/api/network
