# Linux Server Monitoring Center

Enterprise monitoring solution for Linux servers using Grafana, Prometheus, Docker, and Spring Boot.

## Features

- CPU Monitoring
- Memory Monitoring
- Disk Monitoring
- Server Health Status
- Prometheus Metrics
- Alert Rules
- Grafana Dashboards
- Docker Deployment

## Technology Stack

- Java 21
- Spring Boot
- Prometheus
- Grafana
- Docker
- Node Exporter

## Services

Grafana

http://localhost:3000

Prometheus

http://localhost:9090

Monitoring API

http://localhost:8080/api/linux
