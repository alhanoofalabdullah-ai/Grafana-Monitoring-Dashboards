# Project 06 – Kubernetes Cluster Monitoring Dashboard

Enterprise Kubernetes monitoring platform using Grafana, Prometheus, Kubernetes Metrics, Spring Boot, and Docker.

---

## Features

- Cluster Monitoring
- Node Monitoring
- Pod Monitoring
- CPU Monitoring
- Memory Monitoring
- Namespace Monitoring
- Alert Management
- Grafana Dashboards

---

## Technology Stack

- Java 21
- Spring Boot
- Kubernetes
- Prometheus
- Grafana
- Docker

---

## Services

Grafana

http://localhost:3000

Prometheus

http://localhost:9090

Monitoring API

http://localhost:8080/api/cluster
