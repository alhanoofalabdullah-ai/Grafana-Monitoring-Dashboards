# Project 04 – API Performance Monitoring Dashboard

Enterprise API performance monitoring dashboard using Grafana, Prometheus, Spring Boot Actuator, Docker, and custom API metrics.

## Overview

This project monitors REST API performance, availability, latency, error rate, and request volume.

## Features

- API Health Monitoring
- Request Count Tracking
- Response Time Monitoring
- Error Rate Tracking
- Prometheus Metrics
- Grafana Dashboard
- Alert Rules
- Docker Deployment

## Technologies

- Java 21
- Spring Boot
- Spring Boot Actuator
- Micrometer
- Prometheus
- Grafana
- Docker

## Services

Grafana: http://localhost:3000

Prometheus: http://localhost:9090

API Metrics Service: http://localhost:8080/api/metrics

Actuator Metrics: http://localhost:8080/actuator/prometheus
