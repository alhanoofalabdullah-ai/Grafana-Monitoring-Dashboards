# Project 05 – Application Log Monitoring Dashboard

Enterprise log monitoring solution using Grafana, Loki, Promtail, Docker, and Spring Boot.

## Features

- Centralized Log Collection
- Error Monitoring
- Warning Monitoring
- Log Search
- Log Analytics
- Grafana Dashboards
- Loki Log Storage
- Promtail Log Shipping

## Technology Stack

- Java 21
- Spring Boot
- Grafana
- Loki
- Promtail
- Docker

## Services

Grafana

http://localhost:3000

Loki

http://localhost:3100

Application API

http://localhost:8080/api/logs
